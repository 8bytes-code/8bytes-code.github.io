卡win10更新

虽然不知道有没有得用，凑合玩玩也是百度的

禁用windows update

  1. win+r 打开 运行，输入services.msc，回车
  2. 找到windows Update，双击打开点击停止，将启动类型选为“禁用”，点击应用
  3. 切到恢复选项，将默认的重新启动服务改为误操作，点击应用，然后确定

修改组策略

  1. win+r 打开运行，输入gpedit.msc，回车
  2. 在编辑器中，依次展开计算机配置-管理模板-windows组件-windows更新
  3. 找到配置自动更新设置，点击编辑策略设置，将其设置为已禁用，然后点击应用，点击确定
  4. 之后还需要再找到“删除使用所有Windows更新功能的访问权限”，选择已启用，完成设置后，点击应用，点击确定。

禁用任务计划

  1. win+r打开运行，输入taskschd.msc，回车
  2. 依次展开任务计划程序库 -> Microsoft -> Windows -> WindowsUpdate，里面可能有两个或者一个，选中一个点击设置为禁用

修改注册表

  1. win+r 打开运行，输入regedit，回车
  2. 依次展开[HKEY_LOCAL_MACHINE\SYSTEM\CurrentControlSet\Services\UsoSvc]先找到start，点击修改，选择十六进制，值改为4。
  3. 然后找到FailureActions，右击修改二进制数据，将0018那行的第三组数据改为00

结语

具体效果不好说，但是至少看到windows更新受到限制。

目的只是记录，免得下次改不回来

禁用更新的初衷是太频繁了，虽然有些更新是好的补丁，但是有些就不好说了，而且更新不一定就好，也会出问题，所以选个版本卡死就行。不确定的就像我这样记录下来以免下次出问题改回去。
