前言

赋值运算符重载

前言

class Box{
public:
    int x;
    int y;
};

默认的副本构造形式大概就是Box(const Box& box): x{ box.x }, y{ box.y }{ }
这也是为什么初始化的时候，Box b2(b1)是合法的。
